Privacy Policy for Automated Data Entry Chrome Extension

Effective Date: Apr 29 2025

This Privacy Policy explains how the Automated Data Entry Chrome Extension ("the Extension", "we", "our", or "us") collects, uses, and protects your information when you use our tool to extract and structure data from PDFs into formats like CSV, SQL, or JSON.

1. Information We Collect

We collect only the minimum information required to provide and improve our services:

  • Uploaded PDFs: When you upload PDF files, the data is processed via secure API calls. Files may be temporarily transmitted to our servers for text extraction and structuring.
  • Export Preferences: We store non-personal preferences such as your selected export format (CSV, SQL, JSON) to improve user experience.
  • Extension Usage Data: We may collect anonymized metrics on feature usage to enhance functionality and performance.

We do not collect or store:

  • Personal identifiers such as name, email address, or passwords.
  • Your browsing history or unrelated website data.

2. How We Use Your Data

  • To process unstructured data (PDFs) into structured formats.
  • To allow download/export of processed data in your chosen format.
  • To improve product performance and user experience through anonymized analytics.

3. How We Store and Protect Your Data

  • Temporary data processing occurs via secure, encrypted API endpoints.
  • No persistent storage of uploaded PDFs or extracted data unless explicitly enabled for premium or enterprise features.
  • All stored preferences (e.g., export type) are kept locally using Chrome's extension storage API and are never shared.

4. Permissions Explained

  • Storage: Used to save local settings and extraction session state.
  • Tabs: Allows managing extension functionality across opened tabs (e.g., showing results or export options).
  • Host Permissions: Enables access to user-authorized domains where PDF documents are hosted (e.g., cloud storage platforms), with user consent.
  • File Access: Required to read uploaded PDFs directly from your local device.

5. Third-Party Access

We do not sell, trade, or rent your data. Data is only transmitted to our processing APIs, and only for the purpose of fulfilling the extension's core function.

6. Remote Code Execution

This extension does not execute remote code. All executable code is bundled with the extension package and reviewed according to Chrome Web Store policies. Communication with APIs is limited to data transmission and response handling.

7. User Control and Data Deletion

Users can:

  • Remove uploaded files at any time.
  • Clear preferences through Chrome extension settings.
  • Uninstall the extension to delete all locally stored data.

To request deletion of any temporarily stored server data, please contact us at: play3dinc@gmail.com

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in this document with an updated effective date.
