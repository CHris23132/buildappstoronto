Junior JavaScript Developer

Full-time Toronto / Remote

We're looking for a talented Junior JavaScript Developer to join our innovative team at Toronto Innovation Lab. The ideal candidate will have experience developing full applications across various JavaScript frameworks and a passion for creating elegant, efficient code.

Requirements:

  • Experience developing full applications in React
  • Strong knowledge of vanilla JavaScript/TypeScript
  • Experience with Electron for desktop application development
  • Solid background in HTML, CSS, and responsive web development
  • Understanding of web performance optimization techniques
  • Ability to write clean, maintainable, and well-documented code
  • Good problem-solving skills and attention to detail
  • Experience with version control systems (Git)
  • Excellent communication skills and ability to work in a team

Bonus Skills:

  • Experience with mobile app development frameworks (React Native, Flutter)
  • Knowledge of backend technologies (Node.js, Express)
  • Familiarity with CI/CD pipelines
  • Understanding of UI/UX principles
